The only approach that never asks a low-vision user to judge a preset from a 100px thumbnail — the preview is the whole screen, at the size they'll actually watch it. Also the only one where the answer to "does this work at 300% zoom" is trivially yes. Cost: the overlay hides the video while open, the pattern is unfamiliar for a web video player, and it needs real focus-trap and motion discipline to not be disorienting.
